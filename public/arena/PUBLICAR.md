# NT Informática Arena

Este projeto deve ser tratado primeiro como site público da Arena.

## Publicação

Subir a pasta inteira `nt-informatica-app` em uma hospedagem estática com HTTPS, como Vercel, Netlify, GitHub Pages ou servidor próprio.

## Entrada principal

O arquivo principal é `index.html`.

## App instalável

O `manifest.json`, `sw.js` e os ícones continuam no projeto para permitir instalação como PWA no Android depois que o site estiver publicado em HTTPS.
